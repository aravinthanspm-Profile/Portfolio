# 📊 Blinkit Sales Analysis Dashboard

## 🚀 Project Overview

This project is an interactive Power BI dashboard developed using the Blinkit Grocery Sales dataset.

The dashboard provides insights into:

- Total Sales
- Average Sales
- Item Categories
- Outlet Performance
- Fat Content Analysis
- Location-wise Sales
- Establishment Trends

---

## 🛠 Tools & Technologies

- Power BI
- DAX
- Power Query
- Data Modeling
- Data Visualization

## 🔍 Insights

- Tier 3 locations generated maximum sales.
- Regular fat products contributed highest revenue.
- Supermarket Type 1 outlets performed best.
- Fruits and Snack Foods were top-selling categories.
